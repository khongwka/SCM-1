Hello, Version Control!

This is a simple HTML file to demonstrate version control using GitHub.

Follow these steps to use version control:

  1. Initialize a Git repository: git init
  2. Add files to the staging area: git add .
  3. Commit changes: git commit -m "Initial commit"
  4. Create a GitHub repository and add it as a remote: git remote add origin https://github.com/yourusername/your-repo-name.git
  5. Push changes to GitHub: git push -u origin master
